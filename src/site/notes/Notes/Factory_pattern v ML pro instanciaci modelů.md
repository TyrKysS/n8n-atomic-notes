# 1. Popis
Factory a Abstract Factory pattern umožňují zjednodušit tvorbu různých typů objektů, např. recommender modelů, tím, že schovávají komplexitu konstrukce pod společné rozhraní, což usnadňuje přepínání a správu různých modelů během evaluace.
