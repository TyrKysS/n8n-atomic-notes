# 1. Popis
Flyweight pattern minimalizuje spotřebu paměti v recommender systémech tím, že sdílí neměnný společný stav objektů, například metadata filmů, a uchovává uživatelsky specifická data externě, čímž se vyhýbá duplikaci dat pro miliony uživatelů.
