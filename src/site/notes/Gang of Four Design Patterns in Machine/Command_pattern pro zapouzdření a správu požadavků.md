# 1. Popis
Command pattern zapouzdřuje požadavky do objektů, což umožňuje jejich frontování, logování a pozdější opětovné provedení, užitečné v recommender systémech například pro zaznamenávání uživatelských interakcí a jejich asynchronní zpracování či retraining.
