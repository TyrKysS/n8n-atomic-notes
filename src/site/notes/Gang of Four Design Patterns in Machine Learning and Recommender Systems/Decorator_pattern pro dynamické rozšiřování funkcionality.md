# 1. Popis
Decorator pattern umožňuje přidávat nové funkce ke stávajícím recommender objektům dynamicky, např. filtrování nebo vyloučení již zobrazených položek, aniž by bylo potřeba měnit původní kód základního doporučovacího modelu.
