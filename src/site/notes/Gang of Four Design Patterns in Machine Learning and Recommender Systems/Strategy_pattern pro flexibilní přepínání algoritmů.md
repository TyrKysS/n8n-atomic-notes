# 1. Popis
Strategy pattern umožňuje zahrnout různé algoritmy doporučování jako zaměnitelné strategie, což pomáhá flexibilně měnit použité metody v závislosti na kontextu a dostupnosti dat, například řešit cold-start problém dynamickým přepínáním.
