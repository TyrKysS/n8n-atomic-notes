# 1. Popis
Pro přenos a zpracování dat v inteligentních domech se používají otevřené standardy XML a JSON. XML nabízí strukturovaný, validovatelný formát, vhodný pro složitější datovou výměnu, zatímco JSON je lehčí a efektivnější, často používaný v mobilních aplikacích a webových službách.
