# 1. Popis
Návrhový vzor Singleton se používá pro řízení zdrojů, kde je potřeba mít jen jednu instanci daného objektu, například pro správu spojení do databáze, aby se zabránilo nadměrnému otevírání více spojení a tím i možné ztrátě výkonu či nefunkčnosti systému.
