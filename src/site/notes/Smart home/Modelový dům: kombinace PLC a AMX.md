# 1. Popis
Praktický model chytrého domu ukazuje, že efektivní architektura kombinuje nízkoúrovňový PLC (např. Tecomat Foxtrot) pro spínaná I/O a lokální řízení s vyšším centrálním řídicím prvkem (např. AMX NI‑2100) pro integraci EZS, biometriky, kamer a multimédií, přičemž AMX vystupuje jako integrační vrstva a server pro mobilní ovládání (iPad), zatímco PLC zajišťuje spolehlivé řízení světel, žaluzií, měření impulsů elektroměru a regulaci vytápění; tento hybridní přístup kombinuje robustnost průmyslového řízení a flexibilitu uživatelského rozhraní.
