# 1. Popis
Vizuální programování umožňuje uživatelům bez hlubokých programátorských znalostí vytvářet řízení PLC pomocí grafických funkčních bloků (AND, OR, NOT, časovače), což usnadňuje konfiguraci a automatizaci inteligentního bydlení bez nutnosti psaní textového kódu.
